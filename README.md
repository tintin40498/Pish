# 🛡️ Pish - Anti-Phishing Intelligence Platform

**Pish** es una plataforma de detección y bloqueo automático de dominios de phishing. Ideal para PYMEs que quieren proteger a sus empleados sin complicaciones.

## 🚀 Características

- ✅ **Detección automática** de dominios de phishing (11 clasificaciones)
- ✅ **API REST** para integración con firewalls/SIEM
- ✅ **Dashboard web** con estadísticas en tiempo real
- ✅ **Bloqueo en /etc/hosts** automático
- ✅ **Clasificación inteligente** (phishing_login, phishing_verify, etc.)

git clone https://github.com/tintin40498/pish.git
